Documentação transcrita:
Explicação das funções criadas e seus usos:
 - Funções nomeadas:
   - soma(a, b): retorna a soma de dois números;
   - subtracao(a, b): retorna a diferença entre dois números;
   - multiplicacao(a, b): retorna o produto de dois números;
   - divisao(a, b): retorna o resultado da divisão, com verificação para divisão por zero.
 - Funções que manipulam arrays (arrow functions):
   - filtrarPares(numeros): usa filter() para retornar apenas os números pares de um array;
   - calcularMedia(numeros): usa reduce() para somar os valores e calcula a média.
 - Função com callback:
   - executarOperacao(a, b, operacaoCallback): recebe dois números e uma função (ex: multiplicacao) como parâmetro, executando a operação passada.
